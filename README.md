### How to Use

Clone this repository

```shell
git clone https://github.com/Deep062001/preImgAssign.git
```

Move to the repository's root folder

```shell
cd preImgAssign
```

##### For running the react app

Get to the root folder and type this:

```shell
npm install
npm start
```
